Обо мне

Привет! Меня зовут Настя, я начинающий аналитик данных (Junior Data Analyst). Сейчас учусь работать с данными: превращать сырые таблицы в понятные отчёты, находить закономерности и формулировать гипотезы.

В проектах использую Python (Pandas, Statsmodels) для анализа данных, Power BI — для создания интерактивных дашбордов, Excel — для обработки и проверки данных. Также проходила обучение по веб-верстке, поэтому привыкла писать аккуратный код, работать с Git и внимательно относиться к структуре данных.

Мне интересно не просто строить графики, а разбираться, что стоит за цифрами: находить факторы, которые влияют на результат, и замечать аномалии в данных. Сейчас продолжаю развиваться в аналитике данных и ищу команду, где смогу начать карьеру аналитика и применять эти навыки на практике.
