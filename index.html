Обо мне

Привет! Меня зовут Настя, я начинающий аналитик данных (Junior Data Analyst). Сейчас я активно учусь превращать массивы сырых данных в понятные отчёты и рабочие гипотезы.

В проектах использую Python (Pandas, Statsmodels) для анализа зависимостей, Power BI для интерактивных дашбордов и Excel для глубокой обработки данных. Бэкграунд в веб-разработке научил меня писать чистый код, работать с Git и строго следить за качеством данных.

Мне важно находить в цифрах скрытые драйверы роста и аномалии, а не просто строить графики. Постоянно развиваюсь в Data Science и ищу команду, где смогу начать карьеру аналитика и приносить пользу реальному бизнесу.
